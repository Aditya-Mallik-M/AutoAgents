# Basic Example

## Use Cases

### Basic Agent
```sh
export OPENAI_API_KEY=your_openai_api_key_here
cargo run --package basic -- --usecase simple
```
Shows a basic event-driven agent.


### Chaining Multiple Agents
```sh
export OPENAI_API_KEY=your_openai_api_key_here
cargo run --package basic -- --usecase chaining
```
Shows a basic event-driven agent.
